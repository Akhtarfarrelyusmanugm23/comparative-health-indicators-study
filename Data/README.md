# Data Folder

This folder contains the datasets used in this project.

## raw_data
The `raw_data` folder contains original data files downloaded directly from World Bank Open Data. These files have not been modified and serve as the primary source data for this study.

## derived_data
This folder is intended for processed datasets created from the original World Bank files after cleaning, selecting indicators, merging variables, and restructuring the data for analysis.

## Source
All raw data in this project is derived from World Bank Open Data.
